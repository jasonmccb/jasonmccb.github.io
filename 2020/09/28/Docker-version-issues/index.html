    Minimum version for WSL 2:
    Run winver, e.g. 
     Windows 10 Version 2004 (OS Build 19041.329)

    Version 1903 build 18362 needs to join Windows Insider Program

    Solution:
    Cannot use WSL 2 on low versions of Windows 10.
